# QR-Code-Generator

This is a simple QR Code Generator built using HTML, CSS, and JavaScript. The application allows users to enter any text or URL and instantly generate a QR code for it. The QR code is displayed dynamically without reloading the page.

The project uses an online QR code API to generate the QR image and updates it in real time based on user input. It is lightweight, easy to use, and suitable for beginners who want to practice DOM manipulation and event handling in JavaScript.
